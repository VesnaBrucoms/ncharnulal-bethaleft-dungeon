# The Lost Facility of Ncharnulal-Bethaleft

*The Lost Facility of Ncharnulal-Bethaleft* is a large dungeon mod for *Morrowind*.

## Requirements

Official expansions:

* Tribunal
* Bloodmoon

Mods:

* OAAB_Data
* Tamriel_Data (v11)
